#  Gariba Gym / Dev-Vipera SaaS

Plataforma integral de gestión para gimnasios y centros deportivos, diseñada para escalar operaciones, aumentar la retención de clientes y modernizar el control de accesos. Este MVP fue desarrollado como una solución de software como servicio (SaaS) bajo la agencia Dev-Vipera.

##  Características Principales (MVP)

* **Punto de Venta (POS) e Inventario:** Gestión de cobros para membresías (servicios) y productos físicos, con flujo de caja integrado.
* **Control de Accesos (Check-in):** Sistema de validación rápida en recepción.
* **App Ligera para Socios (PWA):** Interfaz móvil con código de acceso, días restantes de membresía y gamificación (rachas de asistencia).
* **Panel de Retención:** Detección automática de socios inactivos o con membresías por vencer, con integración de un clic para WhatsApp.
* **Creador de Rutinas Digitales:** Asignación de rutinas interactivas por días para los socios.
* **Arquitectura Multi-Sucursal:** Soporte para administrar múltiples sedes desde un solo dashboard (Súper Admin).

##  Stack Tecnológico

* **Frontend:** React, Tailwind CSS
* **Backend & Base de Datos:** Firebase (Firestore, Authentication)
* **Arquitectura:** Mobile-first, Progressive Web App (PWA) ready.
